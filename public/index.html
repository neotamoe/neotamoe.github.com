Sample Work

Sci123 Quiz App - http://sci123.herokuapp.com

Solo project created during a two week period during my time at Prime Digital Academy. The quiz app is designed to complement a general education science course at Augsburg College (Sci123: Science of Food and Cooking). This full stack web application includes a mongoDB with over 1000 quiz questions. Questions are sorted by chapter as well as assigned up to 5 tags. Students sign in to quiz themself on course content or submit a new question for inclusion in the quiz (verification done using Passport). Questions can be selected by chapter or by tag/keyword. Application administrator is notified by email when a student flags a question for review or submits a question for inclusion in the database. The administrator can edit, save or delete flagged or newly-submitted questions from the database.

St Croix Valley Interactive Map - http://stcroixmap.herokuapp.com

Interactive map with administrator view/privileges created for client (St Croix Valley Regional Tourism Alliance) for group project at Prime Digital Academy with three other students/contributors. Application utilizes Google Maps API. Map view features area attractions from four categories or suggested trips. Trips show routes from user's current location. Admin view allows adding, editing and deleting of attractions or trips. Admin authentication is done using Firebase. Application was completed over a three week period. I wrote a variety of code, but most involved front-end and client-side code including info window, multiple modals, custom map properties, custom map markers, navigation bar (responsive to screen size), and preparing site for deployment with environment variables.

OMDB Movie API - http://search-and-store-fave-movies.herokuapp.com

This full stack web application was made over a weekend about halfway through my time at Prime (week 5 of 7 in Tier 2). Users search for movies and can select favorite movies to store on the favorites page. This project was done before the OMDB API required a subscription. The currently deployed version displays favorites, but no longer can search with OMDB API.

Angular High Fives - http://shield-high-fives.herokuapp.com

Web application made during in class time while learning and developing comfort with ng-view, services, controllers. Users "high-five" a character and are randomly generated their "skill" and it is compared to the character's randomly generated "skill" level. Each character has a different skill range. Client-side logic compares the skill levels and calculates and displays points.

Server Side Calculator - http://form-over-function-calculator.herokuapp.com

This application was made in week 2 of Tier 2. Calculations are made on server-side to practice server-side logic. My emphasis was on functionality rather than styling.
